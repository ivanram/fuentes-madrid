# Fuentes de Madrid 💧

Web app para encontrar la fuente pública de agua potable más cercana en Madrid.

## Qué hace

- Localiza tu posición y muestra en un mapa la fuente más cercana.
- Ficha de cada fuente: dirección, para quién es (personas / mascotas / mixto) y su estado.
- Ruta a pie hasta la fuente, o guía en **realidad aumentada** con la cámara y la brújula del móvil.
- Filtros por tipo de fuente y favoritas.
- Varios temas de mapa y de color, español/inglés, modo claro/oscuro.
- Funciona sin conexión como **PWA** instalable; la ubicación nunca sale del móvil.

## Datos

2.236 fuentes operativas, con datos oficiales del [Ayuntamiento de Madrid](https://datos.madrid.es/dataset/300051-0-fuentes) (CC BY 4.0), actualizados automáticamente cada semana.

## Tecnología

HTML/CSS/JS sin frameworks ni build step, con [Leaflet](https://leafletjs.com/) para el mapa.
